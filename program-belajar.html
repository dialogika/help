    FAQ Program Belajar
  • Program apa saja yang disediakan di Dialogika ?

    Dialogika menyediakan berbagai program pembelajaran untuk anak-anak dan dewasa. Beberapa program yang populer untuk dewasa termasuk:

    • - Basic Play (Online via Zoom)
    • - Basic Plus (Online via Zoom)
    • - Basic Private (Online via Zoom)
    • - First Class (Offline/tatap muka)
    • - First Private (Offline/tatap muka)

    Sedangkan untuk anak-anak, program-program yang banyak diminati meliputi :

    • - Kids Play (Online via Zoom)
    • - Kids Plus (Online via Zoom)
    • - Kids Private (Online via Zoom)
    • - First Kids (Offline/tatap muka)
  • Apa bedanya Basic Plus dan First class ?

    Basic Plus adalah program pengembangan diri dan pelatihan public speaking selama 1 (satu) bulan, dilaksanakan secara online melalui Zoom, dengan peserta antara 1 hingga 10 orang.

    First Class adalah program serupa yang lebih intensif, berlangsung selama 2 (dua) bulan dengan 10 (sepuluh) kali pertemuan secara offline/tatap muka. Program ini tersedia di Yogyakarta, Tangerang, dan Surabaya.
